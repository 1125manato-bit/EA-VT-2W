# EA VT-2W

**EMU AUDIO** - Clean / Hi-Fi / Transparent Saturation Processor

---

## プラグインモデル・キャラクター

### コンセプト: **「磨く」サチュレーション**

EA VT-2Wは、従来のサチュレーションプラグインとは異なるアプローチで設計されています。
「歪ませる」「温かくする」のではなく、**「音を磨き上げる」**ことを目的としています。

| 特性 | 説明 |
|------|------|
| **Pure（純粋）** | 濁りのない、クリスタルのようなサウンド |
| **Clear（明瞭）** | 高解像度で細部まで聴こえる |
| **Transparent（透明）** | 原音のキャラクターを損なわない |
| **Hi-Fi（高忠実度）** | レンジの広さを維持したまま処理 |

### 想定モデル: **ハイエンド・ソリッドステート・プリアンプ**

VT-2Wは、以下のような機材をイメージしてモデリングされています：
- 高級スタジオコンソールのサミングアンプ
- クリーン志向のソリッドステート・プリアンプ
- マスタリンググレードのライン・アンプ

**真空管（Tube）やテープ（Tape）の色付けを意図的に排除**し、原音の純度を最大限に保ちながら、アナログ機材特有の「存在感」と「輪郭」のみを付加します。

## 技術的特徴

### 1. ハイファイ・サチュレーション回路
```
処理フロー:
入力 → 非対称S字カーブ → 2次倍音付加 → 3次倍音補正 → トランジェント強調 → 出力
```

- **非対称S字カーブ**: 緩やかな圧縮感を与え、デジタル的なピークを音楽的に制御します。
- **倍音生成**: 偶数次（2次）倍音を主体に付加することで、音に豊かな「太さ」と「密度」を与えます。
- **周波数特性**: ワイドレンジかつ完全にフラット。低域・高域の明瞭度を一切損ないません。

### 2. トランジェント・エンハンスメント
- アタック成分を検出し、**音の輪郭を明瞭化**
- コンプレッサーとは逆のアプローチ：ダイナミクスを**保護・強調**
- 埋もれがちな音を前に出す効果

### 3. 自動ゲイン補正
- Driveを上げても音量が大きくなりすぎない
- レベルマッチングが容易で、A/B比較がしやすい

## パラメータ詳細

### DRIVE (0.0 - 100.0)

| 範囲 | キャラクター | 用途 |
|------|--------------|------|
| 0-20 | ほぼ透明 | マスタリング、繊細な素材 |
| 20-50 | わずかな質感 | ボーカル、アコースティック楽器 |
| 50-80 | 輪郭の明瞭化 | ドラム、ベース、ミックスバス |
| 80-100 | 存在感の強調 | リード楽器、目立たせたい素材 |

### MIX (0.0 - 100.0)
- **0%**: 完全ドライ（エフェクトなし）
- **50%**: パラレル処理（繊細なブレンド）
- **100%**: 完全ウェット（フル効果）

## 他のVTシリーズとの違い

| 製品 | キャラクター | 用途 |
|------|--------------|------|
| **VT-2B** | Glue / Tight / Controlled | まとめる・整える |
| **VT-2W** | Pure / Clear / Transparent | 磨く・輪郭を出す |

VT-2Wは、VT-2Bの「まとめる」アプローチとは対照的に、**個々の音の独立性を保ちながら全体のクオリティを上げる**ためのツールです。
